# Job Portal Frontend
 
 This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.
 ## 🌟 Overview
 This is the frontend of the Job Portal web app, built using **React.js** and **CSS** for styling. It connects with the backend to enable job seekers, employers, and admins to interact seamlessly.
 
 Currently, two official plugins are available:

 ## 🔧 Tech Stack
 - **Framework:** React.js
 - **Styling:** CSS
 - **API Handling:** Axios
 
 ---
 
 ## 🚀 Features
 
 ### 🎯 Job Seeker Features
 - User Registration & Login

 - Apply for Jobs
 
 ### 💼 Employer Features
 - Employer Registration & Login
 - Post Job Openings
 - View Applications
 
 ### 🛠️ Admin Panel
 - Dashboard
 - User Management

 ## 🔥 Setup Instructions
 
 ### 🔧 1. Clone the Repository
 ```bash
 git clone https://github.com/manglamonika/jobLearning.git
 cd jobPortal
 ```
 
 ### 📦 2. Install Dependencies
 ```bash
 npm install
 ```
 
 ### 🔥 3. Create Environment File
 👉 **Create a `.env` file** and add:
 ```
 VITE_BACKEND_URL=https://your-backend-url.com
 ```
 
 ### 🚀 4. Run the Frontend
 ```bash
 npm run dev
 ```
